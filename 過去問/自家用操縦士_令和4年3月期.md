### 航空通信

**問1**

航空機用救命無線機（ELT）について誤りはどれか。

- [ ] ① ELTは不時着した航空機の位置を発見する目的で作られた発信機である。
- [ ] ② 406MHzで捜索救難衛星に識別符号を含むデータを送信している。
- [ ] ③ 遭難通信を知った航空機のパイロットは、救難活動に参加していない場合、援助が実施されていることが明白になった後も遭難通信の周波数に追随してモニターしなければならない。
- [ ] ④ パイロットは、意図しないELTの発信を行った場合は、リセット操作を行い、直ちにその旨をRCC（救難調整本部）に通報すること。

:::spoiler 正答
3

```
用語: 航空機用救命無線機（ELT）＝不時着時に自動または手動で遭難信号（406MHz及び121.5/243.0MHz）を発信し、機体の位置特定を助ける装置。
① ELTは不時着した航空機の位置を発見する目的で作られた発信機である … 正。ELTの目的として正しい。
② 406MHzで捜索救難衛星に識別符号を含むデータを送信している … 正。COSPAS-SARSAT衛星へ識別符号付きデータを送信する。
③ 救難活動に参加していない場合、援助が実施されていることが明白になった後も遭難通信の周波数に追随してモニターしなければならない … 誤。援助の実施が明白になった後はモニターを継続する必要はなく、追随義務はない。
④ 意図しないELTの発信を行った場合はリセット操作を行い、直ちにRCCに通報する … 正。誤発信時の措置として正しい。
```
:::

**問2**

捜索救難の発動基準「警戒の段階」について誤りはどれか。

- [ ] ① 第一段通信捜索で当該航空機の情報が明らかで無い場合
- [ ] ② 第一段通信捜索開始後30分を経ても当該航空機の情報が明らかでない場合
- [ ] ③ 航空機が着陸許可を受けた後、予定時刻から5分以内に着陸せず当該航空機と連絡がとれなかった場合
- [ ] ④ 位置通報または運航状態の通報が予定時刻から30分過ぎてもない場合

:::spoiler 正答
4

```
用語: 捜索救難は不確実の段階・警戒の段階・遭難の段階の3段階で判定される。
① 第一段通信捜索で当該機の情報が明らかでない場合 … 正。警戒の段階に該当する。
② 第一段通信捜索開始後30分を経ても情報が明らかでない場合 … 正。警戒の段階に該当する。
③ 着陸許可を受けた後、予定時刻から5分以内に着陸せず連絡がとれない場合 … 正。警戒の段階に該当する。
④ 位置通報または運航状態の通報が予定時刻から30分過ぎてもない場合 … 誤。これは「不確実の段階」に該当し、警戒の段階ではない。
```
:::

遭難および緊急時の通信で誤りはどれか。
