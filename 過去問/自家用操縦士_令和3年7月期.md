### 航空通信

**問1**

航空交通業務のうち、航空機に対して提供される飛行情報業務の内容で誤りはどれか。

- [ ] ① 管制業務
- [ ] ② 航法援助施設の運用状況
- [ ] ③ 飛行場およびその附属施設の状況
- [ ] ④ その他運航の安全のために必要な情報

:::spoiler 正答
1

```
用語: 飛行情報業務＝航空機の安全かつ効率的な航行に有用な情報を提供する業務。
① 管制業務 … 誤。管制業務は航空機相互の間隔設定等を行う別の業務であり、飛行情報業務として提供される内容ではない。
② 航法援助施設の運用状況 … 正。提供内容に含まれる。
③ 飛行場およびその附属施設の状況 … 正。提供内容に含まれる。
④ その他運航の安全のために必要な情報 … 正。提供内容に含まれる。
```

**問2**

捜索救難の措置基準について誤りはどれか。

- [ ] ① 拡大通信捜索で当該航空機の情報が明らかでない場合は不確実の段階である。
- [ ] ② 航空機の航行性能が悪化したが不時着のおそれがある程でない旨の連絡があった場合は警戒の段階である。
- [ ] ③ 位置通報または運航状態の通報が予定時刻から30分過ぎてもない場合は不確実の段階である。
- [ ] ④ 航空機がその予定時刻から30分（ジェット機にあっては15分）過ぎても目的地に到着しない場合は不確実の段階である。

:::spoiler 正答
1

```
用語: 捜索救難の措置基準＝不確実の段階・警戒の段階・遭難の段階の3段階で判定される。
① 誤。拡大通信捜索によっても当該機の情報が明らかでない場合は「遭難の段階」に該当する。
② 正。航行性能が悪化したが不時着のおそれがある程ではない場合は警戒の段階。
③ 正。位置通報・運航状態の通報が予定時刻から30分過ぎてもない場合は不確実の段階。
④ 正。到着予定時刻から30分（ジェット機は15分）過ぎても未着の場合は不確実の段階。
```

① 自機の呼出符号 … 正。通報事項である。
② 遭難信号を発信している航空機の呼出符号 … 誤。遭難信号からは発信元の航空機を識別できず、通報事項に含まれない。

④ 受信終了地点・高度・時刻 … 正。通報事項である。

**問4**

有視界飛行方式における通信機故障の対処について誤りはどれか。

- [ ] ① 受信機のみの故障が考えられるので一方送信を行う。
- [ ] ② VMCを維持して安全に着陸できる最寄りの飛行場に着陸する。

**問5**

航空情報の説明で正しいものはどれか。

- [ ] ① 航空情報サーキュラー：情報の性質又は時期的な理由から航空路誌への掲載又は（AIC）       ノータムの発行に適さないが、航空情報として公示する必要があるもの
- [ ] ② ノータム      ：航空路誌改訂版または航空路誌補足版の情報のうち重要なもの
- [ ] ③ 航空路誌改訂版   ：AIPの一時的変更に係る情報（有効期間が３ヶ月以上のもの等）を掲載
- [ ] ④ 航空路誌補足版   ：AIPの永続的変更に係る情報を掲載

1

③ 誤。航空路誌改訂版はAIPの「永続的」変更に係る情報を掲載する。
④ 誤。航空路誌補足版はAIPの「一時的」変更（有効期間3ヶ月以上のもの等）を掲載する。③と④の説明が入れ替わっている。

**問12**

高度計規正方式について誤りはどれか。

- [ ] ① 離陸前にタワーから提供されたQNHの値をセットした。
- [ ] ② 宮崎空港の管制圏を通過するため宮崎空港のQNHをセットした。
- [ ] ③ 瀬戸内海の海面上を飛行するのでQFEをセットした。
- [ ] ④ 平均海面上14,000フィートを飛行するのでQNEをセットした。

:::spoiler 正答
3

```
用語: QNH＝当該地の海面気圧に規正し平均海面からの高度を指示させる方式。QFE＝飛行場面で高度計が0を指す規正。QNE＝29.92inHgをセットする方式（14,000ft以上）。
① 正。離陸前はタワー等から入手したQNHをセットする。
② 正。他飛行場の管制圏通過時は当該飛行場のQNHをセットする。
③ 誤。QFEは飛行場での離着陸時に滑走路からの高さを示す規正であり、海面上の飛行で用いるものではない。14,000ft未満の飛行はQNHによる。
④ 正。14,000ft以上の飛行ではQNE（29.92inHg）をセットする。
```

**問13**

試験通信において、受信の感明度を通報する場合の「困難だが聞きとれる」を意味する数字で正しいものはどれか。

- [ ] ① ２
- [ ] ② ３
- [ ] ③ ４
- [ ] ④ ５

通信要領の文字と数字の言い表し方で誤りはどれか。
