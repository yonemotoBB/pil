# 🌤 気象

## 気温減率の種類

高度が上がるにつれ気温が下がる割合（℃/高度）。**周囲の大気の温度分布を表すもの**（環境・標準）と、**空気塊自身が上昇したときの温度低下を表すもの**（断熱）を区別する。

| 種類 | 定義 | 代表値 |
|---|---|---|
| 環境気温減率（ELR／実際の気温減率） | 実際の大気で高度が上がるにつれ気温が下がる割合。場所・時間・気象で変動し、逆転（高度とともに気温上昇＝逆転層）もある | 変動（実測値） |
| 標準大気の気温減率 | 国際標準大気（ISA）が定めた対流圏の気温減率 | **約2℃/1,000ft**（<span style="color:#999">1.98℃/1,000ft</span>＝6.5℃/1,000m） |
| 乾燥断熱減率（DALR） | **未飽和（乾燥）**の空気塊が断熱的に上昇するときの温度低下率。ほぼ一定 | **約1℃/100m**（<span style="color:#999">≒3℃/1,000ft＝9.8℃/1,000m</span>） |
| 湿潤断熱減率（MALR／SALR） | **飽和**した空気塊が断熱上昇するときの温度低下率。凝結の潜熱放出のため乾燥より小さく、気温が高いほど小さい | **約0.5℃/100m**（おおむね0.3〜0.9℃/100m） |

**ポイント**
- 断熱減率（乾燥・湿潤）は「空気塊自身」の温度変化、環境・標準減率は「周囲の大気」の温度分布。混同しない。
- 大小関係は原則 **湿潤断熱減率 ＜ 乾燥断熱減率**（飽和すると潜熱で冷えにくい）。

**安定度との関係**（環境気温減率と断熱減率の比較）
- 環境減率 ＜ 湿潤断熱減率 → **絶対安定**
- 環境減率 ＞ 乾燥断熱減率 → **絶対不安定**
- 湿潤断熱減率 ＜ 環境減率 ＜ 乾燥断熱減率 → **条件付（き）不安定**

## 霧の種類

いずれも空気が露点まで冷える／水蒸気が加わって**飽和**することで発生。冷やし方・水蒸気の与え方で分類する。

| 種類 | 発生機構 | 発生しやすい条件 | 備考 |
|---|---|---|---|
| 放射霧 | 夜間の**放射冷却**で地表付近の空気が露点まで冷える | 晴天・弱風・湿潤な夜、内陸 | 日の出後の日射で消散。曇天・強風では発生しにくい |
| 移流霧 | **暖湿な空気**が冷たい地面・海面上を移動し下から冷やされる | 適度な風があっても持続、海上・沿岸 | 海霧の主因 |
| 蒸気霧（蒸発霧） | 冷たい空気が**暖かい水面**上に来て、水面からの蒸発で飽和 | 寒気が暖水面上、晩秋〜冬 | 川霧・けあらし |
| 滑昇霧 | 湿った空気が**山の斜面を上昇**し断熱冷却で飽和 | 風上側の山腹・斜面 | 上昇が続く限り持続 |
| 前線霧（降水霧） | 温暖前線などの**降水（雨）**が下層の冷たい空気中で蒸発し飽和 | 前線付近 | しとしと雨の中で発生 |

**ポイント**
- 放射霧＝地面を冷やす／移流霧＝暖湿空気が冷面へ移動／蒸気霧＝暖水面から水蒸気供給、と機構で区別。
- 放射霧は「風が弱く晴れた夜」、蒸気霧は逆に「冷たい空気が暖かい水面上」がキーワード。

## 着氷（氷）の種類

大気中の水から物体・機体に氷が付く現象。**過冷却水滴**（0℃以下でも液体の水滴）が付着・凍結するもの（雨氷・粗氷・樹氷）と、**水蒸気が液体を経ず直接凍る＝昇華**するもの（樹霜）に分かれ、気象学では次の**4種類**に分類する。

| 種類 | でき方 | 外観・性質 | 密度・硬さ |
|---|---|---|---|
| 雨氷（glaze／クリアアイス） | 過冷却の雨や**大きな**過冷却水滴がゆっくり凍結し層状に付着 | 透明・なめらか、付着力が強く重い、**除去困難** | 最も密で硬い |
| 粗氷（hard rime） | 過冷却水滴が凍結（樹氷と同機構だが凍り方がやや遅い） | **半透明〜透明**に近い塊、雨氷と樹氷の中間 | 中間 |
| 樹氷（soft rime） | **小さな**過冷却水滴が当たった瞬間に凍結、空気を含む | 白色不透明・もろい・粗い表面 | 軽くもろい |
| 樹霜（じゅそう） | 空気中の**水蒸気が液体を経ず直接凍結（昇華）** | 針・板・樹枝状の白い結晶 | 最も軽い |

**ポイント**
- 液体（過冷却水滴）由来＝**雨氷・粗氷・樹氷**／水蒸気由来＝**樹霜**。
- **樹霜・樹氷・粗氷の3つをまとめて「霧氷（むひょう）」**と呼び、これに雨氷を加えて4種類。
- 透明で硬い順は **雨氷 ＞ 粗氷 ＞ 樹氷 ＞ 樹霜**（＝白く・もろく・軽くなる）。着氷は揚力減少・抗力増加・重量増加・計器/操縦への悪影響を招く。

**航空（構造着氷）での扱い**
- 実務では 雨氷＝**クリアアイス**（0〜約−10℃、積乱雲・着氷性の雨、透明で厄介）、樹氷＝**ライムアイス**（約−10〜−20℃、層雲、白くもろい）、両者混在＝**混合氷** に大別。
- 駐機中に機体表面へ水蒸気が昇華した**霜（フロスト）**は、離陸前に必ず除去する。

## 日本周辺の気団

気団は「**寒暖（寒帯/熱帯）**」×「**乾湿（大陸性＝乾燥／海洋性＝湿潤）**」で性質が決まる。日本付近は主に次の4つ。

| 気団 | 性質 | 分類（記号） | 主な季節・特徴 |
|---|---|---|---|
| シベリア気団 | 寒冷・乾燥 | 寒帯大陸性（cP） | **冬**。西高東低、北西季節風。日本海側は降雪、太平洋側は乾いた晴天 |
| オホーツク海気団 | 寒冷・湿潤 | 寒帯海洋性（mP） | **梅雨・秋雨**。冷湿な北東風（やませ）。梅雨前線の**北側** |
| 小笠原気団 | 温暖・湿潤 | 熱帯海洋性（mT） | **夏**。太平洋高気圧、南〜南東季節風、蒸し暑い。梅雨前線の**南側** |
| 揚子江（長江）気団 | 温暖・乾燥 | 熱帯大陸性（cT） | **春・秋**。移動性高気圧として来て乾いた晴天 |

**ポイント**
- **梅雨前線**＝オホーツク海気団（寒・湿）と小笠原気団（暖・湿）がぶつかる停滞前線。
- 季節の主役：冬＝シベリア気団／夏＝小笠原気団／春・秋＝揚子江気団（移動性高気圧と低気圧が交互に通り天気が周期変化）。
- 大陸性＝乾燥、海洋性＝湿潤／寒帯＝寒冷、熱帯＝温暖、と覚えると分類記号（cP・mP・mT・cT）が導ける。

# 📖 法規

## 有視界気象状態（VMC）の基準

計器に頼らず目視で飛行できる気象条件。**飛行視程**と**雲からの距離**で規定（航空法施行規則第5条）。高度3,000mと、管制区・管制圏・情報圏の内／外で基準が変わる。

| 区分 | 飛行視程／地上視程 | 雲からの距離 |
|---|---|---|
| **高度3,000m以上** | 飛行視程 8,000m以上 | <span style="color:#999">水平1,500m・垂直（上方150m／下方300m）</span> |
| **高度3,000m未満**（管制区・管制圏・情報圏 **内**） | 飛行視程 5,000m以上 | <span style="color:#999">水平600m・垂直（上方150m／下方300m）</span> |
| **高度3,000m未満**（管制区・管制圏・情報圏 **以外**） | 飛行視程 1,500m以上 | 航空機を雲から離して飛行できること |
| **管制圏・情報圏内の空港等で離着陸**（地上指定） | **地上視程 5,000m以上** | ― |

**ポイント**
- **高度3,000mが分岐点** → 高高度は視程8,000m。
- **3,000m未満は「管制区等の内か外か」で分岐** → 内は5,000m、外は1,500m。
- 垂直の雲距離は全区分共通で「<span style="color:#999">上方150m／下方300m</span>」（雲との距離の具体値は過去問未出題）。
- 典型問題：令和4年3月期・問1（正答①）。

## 各種証明の有効期間

証明・資格ごとに有効期間が異なる（無期限のものもある）。数字がそのまま問われるので暗記必須。

| 証明 | 条件 | 有効期間 | 備考 |
|---|---|---|---|
| **航空身体検査証明**（自家用＝第二種） | 40歳未満 | **5年** | 起算日は証明が交付された日。年齢は交付日時点で判定 |
| 〃 | 40歳以上50歳未満 | **2年** | |
| 〃 | 50歳以上 | **1年** | |
| **航空英語能力証明** | レベル4 | **4年** | レベルが上がるほど長い |
| 〃 | レベル5 | **6年** | |
| 〃 | レベル6 | **無期限** | |
| **耐空証明**（法第10条） | 原則 | **1年** | 航空運送事業用等は国土交通大臣が定める期間 |
| **技能証明**（操縦士資格） | ― | **無期限**（有効期間の定めなし） | 別途、身体検査証明・技能審査等が必要 |

**ポイント**
- 身体検査証明は「**5→2→1**（年）」＝**40歳・50歳**を境に短くなる、と覚える。
- 英語能力証明は「**4→6→∞**」＝レベル4・5・6。

- 典型問題：耐空証明「有効期間2年」は誤り（正しくは1年）／英語レベル4「2年」は誤り（正しくは4年）。

## 管制圏内等での速度制限（施行規則第179条）

「航空交通管制圏等における速度の制限」。**指示対気速度（IAS）**の上限。高度900mを境に、900m以下ではさらに発動機の種類で分かれる。いずれも**国土交通大臣の許可を受けた場合を除く**。

| 区分 | 指示対気速度の上限 |
|---|---|
| **管制圏内**（原則・高度900m超） | **250kt 以下**（発動機の種類を問わず） |
| 高度**900m以下**の管制圏等・**ピストン**発動機装備機 | **160kt 以下** |
| 高度**900m以下**の管制圏等・**ピストン以外**（タービン等） | **200kt 以下** |

**ポイント**
- 大枠は「管制圏内＝**250kt**」。
- **高度900m以下**になると厳しくなり、**ピストン160kt／その他200kt**。
- ひっかけ：900m以下のピストン機を「200kt」とするのは誤り（正しくは160kt。200ktはピストン以外）。
- 数値はすべて**指示対気速度（IAS）**で規定。

## 救急用具の点検期間

航空法施行規則第151条。用具ごとに点検期間が **60日・180日・12月** のいずれかに決まっており、数字の入れ替えがそのまま誤答選択肢になる。

| 救急用具 | 点検期間 |
|---|---|
| 非常信号灯、携帯灯及び防水携帯灯 | **60日** |
| 救急箱 | **60日** |
| 非常食糧 | **180日** |
| 救命胴衣（これに相当する救急用具）及び救命ボート | **180日** |
| 航空機用救命無線機（ELT） | **12月**（12箇月） |

**ポイント**
- **60日組＝非常信号灯・携帯灯・防水携帯灯・救急箱**（灯火類＋救急箱）。
- **180日組＝非常食糧・救命胴衣・救命ボート**（水/生存系）。
- **ELTだけ「12月」**（24月は誤り）。
- ひっかけ：救命胴衣・救命ボートを「90日」とするのは誤り（正しくは180日）。

## 航空機に備え付ける書類（法59条）

航空機に備え付けが義務付けられる書類。**法本文に明記された書類**と、**その他国土交通省令で定める書類**の区分がそのまま問われる。

**① 法本文に明記された書類**
- 航空機登録証明書
- 耐空証明書
- 航空日誌（搭載用航空日誌）

**② その他国土交通省令で定める書類**
- 運用限界等指定書
- 飛行規程
- 飛行の区間・飛行の方式その他飛行の特性に応じた適切な航空図

**ポイント**
- 航空日誌は**①法本文**の書類（②「その他省令で定める書類」ではない）。
- **無線業務日誌**は電波法に基づく書類であり、法59条の備付け書類には**含まれない**。

## 飛行規程の記載事項（規則5条の4）

飛行規程＝耐空証明とともに指定される、当該航空機の運用に関する文書。主な記載事項：
- 航空機の概要
- 通常の場合における各種装置の操作方法
- 非常の場合に取るべき各種装置の操作その他の措置
- **航空機の騒音**に関する事項（「発動機の騒音」ではない）
- 運用限界・重量重心範囲・性能 等

## 機長の報告義務（法76条）

航空事故・重大インシデント等を国土交通大臣に報告する義務。報告の対象となる例：
- 航空機の墜落・衝突・火災、**地上での他機との接触（衝突）**
- 航行中に**航空保安施設の機能の障害**を知ったとき　等
- ※**他機の墜落を「無線電話で知った」だけ**は報告義務の対象外（自ら目撃・関与した事象が対象）

## 航空英語能力証明が必要な航行（法33条）

- 対象の航空機：**飛行機・回転翼航空機**（飛行船は対象外）
- 本邦内と本邦外の地点間の航行に必要
- **本邦を発し、着陸せず本邦外を通過して本邦に戻る航行**でも必要（大臣が不要と認めたものを除く）
- 有効期間：レベル4＝4年／5＝6年／6＝無期限（→「各種証明の有効期間」表）

## フライトプランに書く記号（第8項「飛行方式および飛行の種類」）

第8項は「**飛行方式（1文字）＋飛行の種類（1文字）**」を記入する。過去問の選択肢・解説に出た記号のみをまとめる。取り違え（V↔Y↔Z、X↔Z）がそのまま誤答になる。

### 飛行方式（Flight rules）

| 記号 | 意味 |
|---|---|
| **I** | 全区間 IFR（計器飛行方式） |
| **V** | 全区間 VFR（有視界飛行方式） |
| **Y** | **IFR**で出発し、途中から**VFR**へ変更 |
| **Z** | **VFR**で出発し、途中から**IFR**へ変更 |

### 飛行の種類（Type of flight）

| 記号 | 意味 |
|---|---|
| **S** | 定期航空運送 |
| **N** | 非定期（不定期）の航空運送事業 |
| **G** | 一般航空（航空機使用事業・訓練・試験・空輸・自家用機など） |
| **M** | 軍用機の飛行 |
| **X** | その他（上記以外） |

**ひっかけポイント**
- **VFRのみ＝「V」**。「Y」ではない（Y＝IFR→VFR、Z＝VFR→IFR）。
- **「その他」＝「X」**。「Z」は飛行方式（VFR→IFR）の記号であり、飛行の種類には用いない。
- 自家用機の飛行は飛行の種類「**G**（一般航空）」。
- 覚え方：飛行方式は **I / V** が全区間、**Y / Z** が途中変更（アルファベット後半＝混在）。

## 飛行計画により明らかにすべき事項（規則203条1項・VFR）

有視界飛行方式の飛行計画で明らかにしなければならない主な事項。**正確な表現**がそのまま問われる。

- 航空機に装備する（使用する）無線設備
- 出発地及び**移動開始**予定時刻
- 巡航高度及び航路（経路）
- 巡航高度における**真対気速度**（巡航速度）
- 最初の着陸地及び離陸後その**上空に到達するまで**の所要時間
- **持久時間で表した燃料搭載量**（＝その燃料で飛行を継続できる時間）

**正確な表現（ここが問われる）**
- 「離陸予定時刻」ではなく **移動開始予定時刻**
- 「着陸するまでの所要時間」ではなく **上空に到達するまでの所要時間**
- 「予想対地速度」ではなく **巡航高度における真対気速度**
- 「消費燃料量」ではなく **持久時間で表した燃料搭載量**

## 空域のクラス分け（クラスA〜D）

福岡FIRは、ICAO標準に従い **クラスA〜E の5つの管制空域** と **クラスG の非管制空域** に分類される。**クラスA〜EをIFRで飛行する場合は、常時、管制機関との通信設定と管制許可が必要**。VFRで飛べるかどうかと、どの機同士に管制間隔（セパレーション）が設定されるかで区別する。

| クラス | 飛行方式 | 管制間隔（セパレーション） | 備考 |
|---|---|---|---|
| **A** | **IFRのみ**（VFR不可） | 全IFR機間に設定 | 最も厳格 |
| **B** | IFR・VFR **両方可** | **すべての航空機間**に設定（VFR相互も） | VFRにも間隔を付与 |
| **C** | IFR・VFR **両方可** | IFR相互＋IFR–VFR間に設定（**VFR相互は交通情報**） | 「VFRのみに限定」は誤り |
| **D** | IFR・VFR **両方可**（＝航空交通管制圏） | **IFR相互のみ**。VFR機には交通情報（**SVFR機を除き管制間隔なし**） | クラスD＝管制圏 |

**ポイント**
- **クラスA＝IFR専用**（VFR不可）。それ以外（B・C・D）はVFRも飛べる。
- 管制間隔の手厚さは **B ＞ C ＞ D**（Bは全機、Cは一部、Dは基本IFRのみ）。
- **クラスD＝航空交通管制圏**。VFR機には管制間隔を設定しない（SVFRを除く）。
- ひっかけ：「クラスCはVFRのみに限定」「クラスBはVFRのみに限定」は誤り（いずれもIFR・VFR両方可）。

## 指向信号灯（ライトガン）の合図

無線が使えない場合等に、管制塔が**色と光り方**で指示を伝える信号灯（飛行場管制業務が行われている空港で使用）。**同じ色でも飛行中と地上で意味が異なる**のが最大のひっかけ。

| 信号（色・光り方） | 飛行中の航空機 | 地上の航空機 |
|---|---|---|
| **緑色の不動光** | 着陸を許可する（着陸支障なし） | 離陸を許可する |
| **緑色の閃光** | 着陸のため飛行場に戻れ（着陸許可は追って発出） | 地上走行してよい |
| **赤色の不動光** | 他機に進路を譲り場周飛行を続けよ | 停止せよ |
| **赤色の閃光** | 着陸してはならない（飛行場使用不能） | 走行区域から離れよ（滑走路の外へ出よ） |
| **白色の閃光** | 着陸してこの飛行場の出発点に帰れ | 飛行場の出発点に帰れ |
| **赤色と緑色の交互閃光** | 危険につき注意せよ | 危険につき注意せよ |

**ポイント**
- **緑＝許可／赤＝禁止・停止／白＝出発点に帰れ／赤緑交互＝注意せよ**。
- 飛行中の **赤色「不動光」＝場周を続けよ（進路を譲れ）**、赤色「**閃光**」＝**着陸してはならない**。不動光と閃光を混同しない。
- 地上の **赤色閃光＝滑走路の外へ出よ**、緑色閃光＝走行してよい。
- 指向信号灯は**飛行場管制業務が行われている空港で使用**（行われていない空港では使わない、は正しい）。

## 捜索救難の段階（遭難の段階）

捜索救難の措置は、緊急の度合いにより **不確実 → 警戒 → 遭難** の3段階で判定される（切迫度が上がる順）。

| 段階 | 英名 | 状態 | 代表的な判定基準 |
|---|---|---|---|
| **不確実の段階** | INCERFA | 航空機・乗員の安全に**疑い**が生じた初期段階 | 予定時刻から**30分**（ジェット機は**15分**）過ぎても到着しない／位置通報がない 等 |
| **警戒の段階** | ALERFA | 航空機・乗員の安全に**不安**が持たれる状態 | 不確実の段階の措置後も安否が確認できない／航行性能が悪化したが不時着のおそれある程ではない旨の連絡 等 |
| **遭難の段階** | DETRESFA | 重大かつ**急迫の危険**にさらされているおそれが確実な最終段階 | 搭載燃料が枯渇/安全な到着に不十分／不時着しようとしている／航行性能が不時着のおそれある程に悪化／更に広範囲な通信捜索によっても情報が得られない 等 |

**ポイント**
- 切迫度は **不確実 ＜ 警戒 ＜ 遭難**。英名 INCERFA→ALERFA→DETRESFA の順。
- **30分（ジェット機15分）**は「**不確実の段階**」の基準。遭難の段階に「30分」という時間の定めはない。
- **搭載燃料が枯渇/安全到着に不十分／不時着**は「**遭難の段階**」。
- ひっかけ：燃料枯渇を「不確実／警戒」とするのは誤り（＝遭難の段階）。

## 🔑 気象：間違えた論点の総まとめ

気象の誤答14問から、覚えるべき正しい内容だけを抽出。

- **ISA 10,000ftの気温**：約−5℃（15℃−2℃/1,000ft×10）
- **ISA 対流圏の気温減率**：0.65℃/100m＝約2.0℃/1,000ft（6.5℃は1,000m当たりの値）
- **コリオリの力**：北半球で運動方向の右直角（北向きの流れ→東向き）。風速に比例し、緯度が高いほど大（sin緯度）。上層は地衡風（気圧傾度力と釣り合う）
- **停滞前線**：秋雨前線＝夏から秋への移行期／梅雨前線＝春から盛夏
- **山岳波でできる雲**：レンズ雲・笠雲・ローター雲（乱層雲は無関係）
- **風速換算**：1kt≒0.51m/s → 10kt≒約5m/s
- **METAR記号 VC**：Vicinity＝飛行場にはないが周辺に観測される
- **下層悪天予想図**：雲底・雲頂は**海抜高度**で表示（フライトレベルではない）。対象は約**15,000ft**まで／**3時間毎・1日8回**発表／6領域（北海道・東北・東日本・西日本・奄美・沖縄）
- **気圧高度計**：規正した高度計は基準気圧面から機外気圧までの距離を標準大気を物差しとして測る。指示高度が一定でも、気温・気圧が標準大気と異なれば真高度は変化する
- **大気の安定**：上昇させた空気塊が周囲より低温（重い）＝安定／周囲より高温＝不安定
- **放射霧**：よく晴れて風の弱い夜に放射冷却で発生（曇天・強風では発生しにくい）
- **海陸風**：海風（日中）は陸風より規模・風速が大きい。局地風でコリオリの影響は小さい
- **台風**：進行方向右側＝危険半円。中心が観測点の北側を東進する場合、観測点は右側に入り風向は時計回り（南西→西→北西）。下層は反時計回りで吹き込み、上層は時計回りで吹き出す
- **着氷**：翼・プロペラ上の空気は加速・減圧し、**膨張**による断熱冷却で水分が凝結する（圧縮ではない）。気化器着氷は断熱膨張＋燃料蒸発（蒸発が主因）
- **空気に働く力**：重力・気圧傾度力・コリオリの力が働く（**「水蒸気圧力」という力はない**。水蒸気圧は水蒸気の分圧を表す量）
- **気団の変質**：寒気団が暖かい地表面上に移動＝下層から加熱され**不安定**（対流盛ん・気流荒れ・積雲形・しゅう雨/雷雨）／暖気団が冷たい地表面上に移動＝**安定**（気流滑らか）
- **移動性高気圧**：温暖型＝**背が高い**／寒冷型＝移動が速く通過後すぐ天気が崩れる
- **南岸低気圧**：日本の南岸を発達しながら東進。冬は太平洋側に降雪。**春一番・メイストームは「日本海低気圧」**であって南岸低気圧ではない
- **TAF（運航用飛行場予報）**：国内発表の有効時間は**30時間**

**含まれない**（自機の事故・重大インシデント等が対象）。
:::

**Q8** 法99条で大臣が提供する航空情報に「運航の遅延」は含まれるか。
:::spoiler 正答
**含まれない**。
:::

**Q9** 施行規則209条の2の航空情報に「重大インシデント」は含まれるか。
:::spoiler 正答
**含まれない**（重大インシデントは事故調査・報告制度の対象）。
:::

**Q10** 昼間障害標識の設置対象物件に「ガスタンク」は含まれるか。
:::spoiler 正答
**含まれない**。対象は「高さに比し幅が狭い物件・風力発電（315m以下）・骨組構造」等。
:::

**Q11** 法92条の禁止飛行に「速度を急激に変更する飛行」は列挙されているか。
:::spoiler 正答
**列挙されていない**（無資格者の練習・限定外機・姿勢を頻繁に変える飛行等が対象）。
:::

**Q12** 法59条で「搭載用航空日誌」は「その他省令で定める書類」に当たるか。
:::spoiler 正答
**当たらない**（航空日誌は本文に明記）。省令で定める書類は運用限界指定書・飛行規程・航空図等。
:::

### C. 数値・区分

**Q13** 管制圏内の速度制限は発動機の種類で分かれるか。
:::spoiler 正答
**分かれない**。発動機を問わず**指示対気速度250kt以下**（900m以下は別途ピストン160/その他200）。
:::

**Q14** VFR飛行計画を飛行開始後に通報する場合、どの範囲で速やかに通報するか。
:::spoiler 正答
**出発地を中心とする半径9km以内**。「離陸後5分以内」ではない。
:::

**Q15** 遭難の段階の判定に「30分」という時間の定めはあるか。
:::spoiler 正答
**ない**。30分（未着・連絡途絶）は**不確実の段階**の基準。
:::

**Q16** 受信の感明度「4」の意味は。
:::spoiler 正答
**良好（聞きとれる）**。「完全に聞きとれる」は**5**。
:::

**Q17** 施行規則209条の位置通報で通報するのは「時刻」と何か。
:::spoiler 正答
**高度**（当該地点における時刻及び高度）。「速度」ではない。
:::

### D. 断定ワードの罠（〜のみ／絶対に／必ず／〜できない）

**Q18** 飛行禁止・制限区域は「必ず告示」で定められるか。
:::spoiler 正答
**否**。ノータム等の航空情報でも設定・周知される。
:::

**Q19** 特別VFRは地上視程1,500m未満なら絶対に許可されないか。
:::spoiler 正答
**否**。**回転翼航空機**は支障のない速度で飛行すれば視程1,500m未満でも許可され得る。
:::

**Q20** 飛行中の航空機は飛行計画を変更できないか。
:::spoiler 正答
**否**。飛行中でも管制機関又は運航情報機関に通報して**変更できる**。
:::

**Q21** 滑走路の横断はグラウンド（地上管制）が担当するか。
:::spoiler 正答
**否**。滑走路の横断は滑走路の使用に関わるため**タワー**が担当する。
:::

### E. 気象

**Q22** ISA（国際標準大気）で10,000ftの気温はおよそ何℃か。
:::spoiler 正答
約**−5℃**（15℃−2℃/1,000ft×10）。
:::

**Q23** ISAの対流圏の気温減率は。
:::spoiler 正答
**0.65℃/100m＝約2.0℃/1,000ft**（6.5℃は1,000m当たりの値）。
:::

**Q24** 北半球でコリオリの力は運動方向のどちらに働くか。北向きの流れでは。
:::spoiler 正答
運動方向の**右直角**。北向きの流れには**東向き**に働く。
:::

**Q25** 秋雨前線が現れるのはどの季節の移行期か。
:::spoiler 正答
**夏から秋**への移行期。
:::

**Q26** 山岳波によってできる雲でないものは（レンズ雲／笠雲／ローター雲／乱層雲）。
:::spoiler 正答
**乱層雲**（前線・低気圧に伴う層状雲で山岳波とは無関係）。
:::

**Q27** 10ktは約何m/sか。
:::spoiler 正答
約**5m/s**（1kt≒0.51m/s）。
:::

**Q28** METARの現在天気「VC」の意味は。
:::spoiler 正答
**Vicinity**＝飛行場にはないが飛行場周辺に観測される。
:::

**Q29** 下層悪天予想図の雲底・雲頂高度は何で示すか。
:::spoiler 正答
**海抜高度**（フライトレベルではない）。対象は約15,000ftまで／3時間毎・1日8回／6領域。
:::

**Q30** 機械的に上昇させた乾燥空気塊が周囲より低温になった。安定か不安定か。
:::spoiler 正答
**安定**（周囲より低温＝重い→元の高度へ戻ろうとする）。
:::

**Q31** 放射霧が発生しやすい条件は。
:::spoiler 正答
**よく晴れて風の弱い夜**（放射冷却）。曇天・強風では発生しにくい。
:::

**Q32** 海風と陸風、規模・風速が大きいのはどちらか。
:::spoiler 正答
**海風**（日中の加熱差が大きいため）。
:::

**Q33** 台風の危険半円は進行方向のどちら側か。
:::spoiler 正答
**右側**。中心が観測点の北側を東進すると観測点は右側に入り、風向は時計回り（南西→西→北西）。
:::

**Q34** 翼やプロペラ上を流れる空気が着氷で冷えるのは、圧縮か膨張か。
:::spoiler 正答
**膨張**（加速して減圧＝断熱膨張で冷却し凝結）。「圧縮」は誤り。
:::

### F. 工学

**Q35** ホバリングは対気速度零か、対地速度零か。
:::spoiler 正答
**対気速度零**の飛行状態。有風時は対気速度零でも対地速度は零にならない。
:::

**Q36** 警報灯・注意灯・指示灯の色分け（緊急／注意／安全）は。
:::spoiler 正答
緊急（直ちに修正）＝**赤**／注意＝**こはく色（黄）**／安全＝**緑**。
:::

**Q37** 「その他の指示灯」に青色の使用は義務づけられているか。
:::spoiler 正答
**否**。青色は使用してよいが「使用しなければならない」とはされていない。
:::

**Q38** 空気（風）に働く力として誤りはどれか（重力／気圧傾度力／コリオリの力／水蒸気圧力）。
:::spoiler 正答
**水蒸気圧力**。そのような力はない（水蒸気圧は水蒸気の分圧を表す量）。働くのは重力・気圧傾度力・コリオリの力。
:::

**Q39** 寒気団が暖かい地表面上に移動すると、成層は安定か不安定か。
:::spoiler 正答
**不安定**（下層から加熱され対流が盛ん・気流が荒れ・積雲形の雲）。安定で気流が滑らかなのは暖気団が冷たい地表面上に移動した場合。
:::

**Q40** 移動性高気圧の温暖型は背が高いか低いか。
:::spoiler 正答
**背が高い**。寒冷型は移動が速く、通過後すぐ天気が崩れる。
:::

**Q41** 春一番やメイストームをもたらす典型的な低気圧は。
:::spoiler 正答
**日本海低気圧**（南岸低気圧ではない）。南岸低気圧は冬に太平洋側へ降雪をもたらす。
:::

**Q42** 国内で発表されるTAF（運航用飛行場予報）の有効時間は。
:::spoiler 正答
**30時間**。
:::

**Q43** 下層悪天予想図の対象高度はおよそ何ftまでか。
:::spoiler 正答
約**15,000ft**まで（雲底・雲頂は海抜高度で表示、3時間毎・1日8回、6領域）。
:::
