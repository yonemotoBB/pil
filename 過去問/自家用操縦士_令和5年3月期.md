### 航空気象

**問1**

対流圏における大気について正しいものはどれか。

- [ ] ① 上層は太陽に近いため下層より気温が高い。
- [ ] ② 高度が上がると気圧も上昇する（大きい値になる）。
- [ ] ③ 高度が上がると空気密度も上昇する（大きい値になる）。
- [ ] ④ 上下の気流の対流が盛んに起こり、雲や雨など天気に密接な関係を持っている。

:::spoiler 正答
4

```
用語: 対流圏＝地表から高度約11kmまでの大気の層。気象現象の大部分がここで起こる。
① 誤。対流圏では高度が上がるほど気温は低下する（平均約6.5℃/km）。大気は主に地表からの放射で下から暖められるためで、太陽への距離はほぼ無関係。
② 誤。高度が上がるとその上にある空気の重さが減るため、気圧は低下する。
③ 誤。空気密度も高度とともに減少する。
④ 正。対流圏は上下の対流が盛んで、雲や雨など天気に密接な関係を持つ層である。
```
:::

**問2**

太陽から地球へのエネルギーの伝達方法で正しいものはどれか。

- [ ] ① 伝導
- [ ] ② 対流
- [ ] ③ 放射
- [ ] ④ 反射

:::spoiler 正答
3

```
① 誤。伝導は物質同士の接触による熱の伝達で、真空の宇宙空間では働かない。
② 誤。対流は流体の移動による熱の伝達で、真空の宇宙空間では起こらない。
③ 正。太陽エネルギーは電磁波（放射）として真空の宇宙空間を伝わり、地球に到達する。
④ 誤。反射はエネルギーの伝達方法ではなく、届いた放射の一部が跳ね返される現象である。
```
:::

**問3**

水分は大気中で固体、液体、気体の3つの形態で存在する。この３つの形態について記述したもので正しいものはどれか。

- [ ] ① 雪は液体としての水分である。
- [ ] ② 霧は気体としての水分である。
- [ ] ③ 雲は液体としての水分として存在し、固体としての水分としても存在する。
- [ ] ④ 気体としての水分は蒸発するときに目で見ることができる。

:::spoiler 正答
3

```
① 誤。雪は氷晶、すなわち固体としての水分である。
② 誤。霧は浮遊する微小な水滴、すなわち液体としての水分である。
③ 正。雲は水滴（液体）から成るものと氷晶（固体）から成るものがあり、両方が混在することもある。
④ 誤。気体としての水分（水蒸気）は目に見えない。蒸発は液体が気体になる過程で、見えるのは凝結した水滴である。
```
:::

**問4**

積乱雲について正しいものはどれか。

- [ ] ① ハケで掃いたような雲である。
- [ ] ② 雲底はカナトコ状に広がっていることが多い。
- [ ] ③ 鉛直に著しく発達した塔状の雲である。
- [ ] ④ 略語はCuである。

:::spoiler 正答
3

```
用語: 積乱雲（Cb）＝強い上昇気流によって鉛直に発達した対流雲。雷・強雨・ひょう・突風を伴う。
① 誤。ハケで掃いたような繊維状の雲は巻雲（Ci）である。
② 誤。カナトコ状に広がるのは雲底ではなく雲頂である。
③ 正。積乱雲は鉛直に著しく発達した塔状の雲である。
④ 誤。積乱雲の略語はCb。Cuは積雲の略語である。
```
:::

**問5**

風は地球の表面に対する空気の動きであり、この空気に働く力として誤りはどれか。

- [ ] ① 重力
- [ ] ② 気圧傾度力
- [ ] ③ 地球自転による見かけ上の力（コリオリの力または偏向力）
- [ ] ④ 水蒸気圧力

:::spoiler 正答
4

```
用語: 気圧傾度力＝気圧差により空気を高圧側から低圧側へ動かす力。風の原動力。
① 正。重力は大気に常に働いている力である。
② 正。気圧傾度力は空気を動かす基本の力である。
③ 正。コリオリの力は地球自転による見かけの力で、北半球では運動方向の右向きに働く。
④ 誤。水蒸気圧力という力は空気の運動には働かない。水蒸気圧は大気中の水蒸気の分圧を表す量にすぎない。
```
:::

**問6**

海陸風について誤りはどれか。

- [ ] ① 海岸地方で、日中、海から陸に向かう風を海風、夜間、陸から海に向かう風を陸風という。小規模な局地風であり、コリオリの影響はほとんどなく､風向は気圧傾度と地形に左右されることが多い。
- [ ] ② 日中、地表付近では陸上は海上よりも気圧が低くなり、逆に上空では陸上は海上よりも気圧が高くなる。上空では気圧の関係が反対になるので逆の方向に向かう風（反流）が吹いている。
- [ ] ③ 海陸風の風速は、一般風に比べあまり強くないので、一般風が強いときや悪天の日には観測されない。
- [ ] ④ 海陸風の水平方向、鉛直方向の規模及び風速は、一般的に陸風のほうが海風よりも大きい。

:::spoiler 正答
4

```
① 正。海陸風は小規模な局地風でコリオリの影響はほとんどなく、気圧傾度と地形に左右される。
② 正。日中は陸が強く加熱され、地表付近では陸上の気圧が低く、上空では逆に高くなるため、上空には反流が吹く。
③ 正。海陸風は弱いので、一般風が強い日や悪天の日には現れにくい。
④ 誤。日中の陸の加熱は夜間の冷却より強いため、規模・風速とも海風の方が陸風より大きい。記述が逆である。
```
:::

**問7**

寒気団が暖かい地表面の上に移動した場合の特性についての説明で誤りはどれか。

- [ ] ① 気団は下の方から加熱されて安定な成層になり対流現象はおさえられ、気流は滑らかである。
- [ ] ② 天気は、しゅう雨、雷雨、雹（ひょう）、霙（みぞれ）、にわか雨である。
- [ ] ③ 安定度は不安定気温減率（ほとんど乾燥断熱減率に近い値を示す）である。
- [ ] ④ 雲形は積雲形で、積雲とか積乱雲である。

:::spoiler 正答
1

```
用語: 寒気団＝移動先の地表面より冷たい気団。
① 誤。寒気団が暖かい地表面上に移動すると下層から加熱されて不安定な成層となり、対流が盛んで気流は荒れる。安定で滑らかになるのは暖気団が冷たい地表面上に移動した場合である。
② 正。不安定なため、しゅう雨・雷雨・ひょう・みぞれ等の対流性の天気となる。
③ 正。気温減率は乾燥断熱減率に近い不安定な値を示す。
④ 正。対流により積雲・積乱雲などの積雲形の雲が発生する。
```
:::

**問8**

気団の説明について誤りはどれか。

- [ ] ① 水平方向にほぼ均一な物理的特性（気温と湿度）を持った空気の大きな塊である。
- [ ] ② 広範囲に一様な性質を持つ地表面に空気が長時間滞留し、その地表面の特性を獲得することで形成される。
- [ ] ③ 特有の気団が形成される地域は気団の発現地と呼ばれる。
- [ ] ④ 気団の変質の度合いは、移動速度、経路の地表面の特性等に依存しない。

:::spoiler 正答
4

```
① 正。気団は気温・湿度が水平方向にほぼ均一な大きな空気塊である。
② 正。一様な性質の地表面上に空気が長時間滞留することで形成される。
③ 正。気団が形成される地域を発現地（発源地）という。
④ 誤。気団の変質の度合いは、移動速度や経路の地表面の特性等に依存する。ゆっくり移動するほど、また地表面との温度差が大きいほど変質は大きい。
```
:::

**問9**

前線のもつ一般的性質について誤りはどれか。

- [ ] ① 前線は気圧の低い谷の中に存在することが多い。
- [ ] ② 移動している前線は、前線を境にして気圧の変化傾向が違う。
- [ ] ③ 前線を境にして気温差がある。
- [ ] ④ 前線の速度は暖気内の風速とほぼ一致する。

:::spoiler 正答
4

```
用語: 前線＝性質の異なる二つの気団の境界面（前線面）が地表と交わる線。
① 正。前線は気圧の谷の中に存在することが多い。
② 正。移動する前線の通過前後では気圧の変化傾向が異なる。
③ 正。前線を境にして気温差がある。
④ 誤。前線の移動速度は、寒気内の風の前線に直角な成分にほぼ一致する。暖気内の風速ではない。
```

**問10**

下図は、ある前線を図式化したものである。正しいものはどれか。

**問11**

高気圧に関する説明のうち誤りはどれか。

- [ ] ① 同じ高度で周囲よりも気圧が高いところを高気圧という。地上天気図では閉じた等圧線で囲まれている。
- [ ] ② 北半球において地上では高気圧から外に向かって時計回りに風が吹き出している。
- [ ] ③ 高気圧の中心では空気量が減少し、それを補うように下降気流が生じる。下降気流のあるところには雲はできないので、一般的に高気圧域内では天気がよい。
- [ ] ④ 高気圧の中心近くでは気圧傾度が大きくなれないので、風は強いが穏やかな天気になる。

:::spoiler 正答
4

```
① 正。高気圧は同じ高度で周囲より気圧が高いところで、地上天気図では閉じた等圧線で囲まれる。
② 正。北半球の地上では高気圧から時計回りに風が吹き出す。
③ 正。中心部では下降気流が生じ、雲ができにくく天気は良い。
④ 誤。高気圧の中心近くは気圧傾度が小さいため風は弱く、穏やかな天気となる。「風は強いが」の部分が誤り。
```

**問14**

対流性の気流の説明で誤りはどれか。

- [ ] ① 対流現象により局地的な上昇・下降の鉛直気流により発生するタービュランスに遭遇することがある。
- [ ] ② 耕作地や水面など地表面が異なると熱の放射量が異なるため、不均一に加熱された空気が局所的な循環を形成する。上昇気流があるとそれを補償する下降気流がある。
- [ ] ③ 舗装地や不毛地上では下降気流が起こりやすく、水や広大な植生地域上では上昇気流が起こりやすい。
- [ ] ④ 上昇気流はやがて飽和に達し雲を形成する。こうした雲の頂は対流性気流の上限を示しているが、空気が乾燥していると雲はできない。

:::spoiler 正答
3

```
① 正。対流による局地的な鉛直気流はタービュランスの原因となる。

③ 誤。舗装地や不毛地は加熱されやすく上昇気流が起こりやすい。水面や広大な植生地域は温まりにくく下降気流が起こりやすい。記述が逆である。
④ 正。上昇気流が飽和に達すると雲を形成し、雲頂は対流性気流の上限を示す。空気が乾燥していれば雲はできない。
```

**問15**

初期突風について誤りはどれか。

- [ ] ① 雷雲が襲来する直前、地上と地上付近におこる風向・風速の急変である。
- [ ] ② 雷雲の中の下降流が地面に達し、地表で水平方向に流れを変え四方に吹き出したものである。
- [ ] ③ 風向の変化は180°も変わることがあり、風速も強烈で50～60ktを超えることがある。
- [ ] ④ 下降流は雷雲からの降水が蒸発することによる周囲の空気の昇温と、大粒の雨が周囲の空気を引きずることで起こる熱い気流である。

**問16**

着氷についての記述で誤りはどれか。

- [ ] ① 一般的に着氷は過冷却水滴（雲・雨）中の飛行で、航空機の機体表面に過冷却水滴が衝突したときに起こりやすい。着氷に関する大気の状態で、着氷のリスクは気温、水分、水滴のサイズにより評価することができる。また、気温が0℃付近では、過冷却水滴が機体表面に衝突後、凍結するまでに後方へ広がることから、Clear Iceという危険な着氷が形成されることがある。
- [ ] ② エンジンの気化器はベンチュリー管のような役目をし、燃料と空気から混合ガスを作る。気化器では空気が断続的に冷却するため空気自身の温度が下がり、着氷が発生する。しかし、気化器冷却の最大の原因は液体燃料の蒸発であり、これによる温度の降下がより大きくなるため、着氷が発生する。
- [ ] ③ プロペラへの着氷は、翼型の先端に氷が堆積することで発生する。着氷によりプロペラの効率が減少する。また、釣り合いが失われて起こる振動は、エンジン等他の構造にとって有害である。
- [ ] ④ 晴れた大気中では着氷は起こりにくいが、相対湿度が高く、気温が0℃に近いようなときにはプロペラの羽や翼の上を通る空気の圧縮のため若干の水分が凝結し、着氷が発生することがある。

:::spoiler 正答
4

```
用語: 過冷却水滴＝0℃以下でも凍らずに液体のままの水滴。物体に衝突すると凍結し着氷となる。
① 正。着氷は過冷却水滴の衝突で起こり、0℃付近では凍結までに水滴が後方へ広がって危険なClear Ice（雨氷）を形成する。
② 正。気化器着氷は空気の断熱冷却に加え、液体燃料の蒸発潜熱による温度降下が最大の原因である。
③ 正。プロペラ着氷は効率を低下させ、不釣り合いによる振動は他の構造にも有害である。
④ 誤。このとき着氷が起こるのは、プロペラや翼の上を通る空気の膨張（圧力低下）による断熱冷却で水分が凝結するためであり、「圧縮のため」が誤り。
```
:::

**問17**

煙霧について誤りはどれか。
