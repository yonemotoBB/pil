### 航空通信

**問1**

航空機用救命無線機（ELT）について誤りはどれか。

- [ ] ① ELTは不時着した航空機の位置を発見する目的で作られた発信機である。
- [ ] ② ４０６MHzで捜索救難衛星に識別符号を含むデータを送信している。
- [ ] ③ 遭難通信を知った航空機のパイロットは、救難活動に参加していない場合には、援助が実施されているいないにかかわらず遭難通信の周波数に追随してモニターしなければならない。
- [ ] ④ パイロットは、意図しないELTの発信を行った場合は、リセット操作を行い、直ちにその旨をRCC（救難調整本部）に通報するべきである。

:::spoiler 正答
3

```
用語: 航空機用救命無線機（ELT）＝不時着した航空機の位置発見のため遭難信号を発信する装置。
① ELTは不時着した航空機の位置を発見する目的で作られた発信機である … 正。定義として正しい。
② 406MHzで捜索救難衛星に識別符号を含むデータを送信している … 正。406MHz型は衛星経由で識別符号付きデータを送る。
③ 遭難通信を知った操縦士は援助が実施されているいないにかかわらず遭難周波数に追随してモニターしなければならない … 誤。援助がすでに実施されていることが明らかな場合はモニターの必要はなく、「実施されているいないにかかわらず」は誤り。
④ 意図しないELT発信を行った場合はリセットし直ちにRCCに通報すべき … 正。誤発信時の措置として正しい。
```
:::

**問2**

捜索救難の発動基準「不確実の段階」について誤りはどれか。

- [ ] ① 航空機が困難な状況に遭遇しているとの情報を受けた場合
- [ ] ② 位置通報が予定時刻から30分過ぎてもない場合
- [ ] ③ 当該航空機の搭載燃料が枯渇したかまたは安全に到着するには不十分であると認められる場合
- [ ] ④ 航空機がその予定時刻から30分（ジェット機にあっては15分）過ぎても目的地に到着しない場合

:::spoiler 正答
3

```
用語: 捜索救難の措置基準＝不確実の段階・警戒の段階・遭難の段階の3段階。不確実の段階は機の安全に疑いが生じた初期段階。
① 航空機が困難な状況に遭遇しているとの情報を受けた場合 … 正。機の安全に疑いが生じた場合で不確実の段階に該当する。
② 位置通報が予定時刻から30分過ぎてもない場合 … 正。不確実の段階の基準である。
③ 搭載燃料が枯渇したかまたは安全に到着するには不十分と認められる場合 … 誤。これは最も切迫した「遭難の段階」に該当し、不確実の段階ではない。
④ 予定時刻から30分（ジェット機は15分）過ぎても目的地に到着しない場合 … 正。不確実の段階の基準である。
```

**問7**

有視界飛行方式により飛行する場合の飛行計画書に記載する所要時間について正しいものはどれか。

- [ ] ① 離陸後、目的空港等に着陸するまでの所要時間
- [ ] ② 発動機始動後の地上滑走開始（ブロックアウト）から、最初の着陸地の上空に到達するまでの所要時間
- [ ] ③ 離陸後、目的空港等の上空に到達するまでの所要時間
- [ ] ④ 発動機始動後の地上滑走開始（ブロックアウト）から、最初の着陸地の駐機場で停止する（ブロックイン）までの所要時間

:::spoiler 正答
3

```
VFRの飛行計画書に記入する所要時間は、離陸から目的空港等の上空に到達するまでの予定時間である（ブロック間ではない）。
① 離陸後、目的空港等に着陸するまでの所要時間 … 誤。着陸までではない。
② 発動機始動後の地上滑走開始（ブロックアウト）から最初の着陸地の上空に到達するまでの所要時間 … 誤。起点はブロックアウトではなく離陸である。
③ 離陸後、目的空港等の上空に到達するまでの所要時間 … 正。VFRの所要時間の定義。
④ ブロックアウトからブロックインまでの所要時間 … 誤。これはIFR等の考え方で、VFRの記入方法ではない。
```
