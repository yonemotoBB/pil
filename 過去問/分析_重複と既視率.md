# 過去問の重複分析と既視率 — 過去問学習だけで何点取れるか

対象: 自家用操縦士（飛・回）学科試験 全17期（平成30年7月〜令和5年7月）・2,020問
方法: 全問題を科目ごとに正規化して類似度比較（SequenceMatcher）し、クラスタリング。
- **類題含む**: 題意が同じ（stem類似度 ≥ 0.85）。数値替えの計算問題も含む
- **選択肢まで同一**: 選択肢の内容まで一致（類似度 ≥ 0.92）＝丸暗記で正解可能

再現: `python3 overlap.py`（このフォルダ内）

## 結論

- 2,020問の実質は **409種類**。95%は使い回しで、**1期限りの独自問題は97問（4.8%）** のみ
- 全過去問を学習した場合の期待点: **丸暗記で約84点／解法理解で約93点**（合格ライン70点を超える）
- 学習範囲は**直近2年分（4期）で8割**に達し、それ以降は収穫逓減

## 重複の規模

| 指標 | 値 |
|---|---|
| 総問題数（17期） | 2,020問 |
| 実質的な問題の種類 | 409種類 |
| 複数期で使い回し | 312種類（延べ1,923問 = 95%） |
| 1期限りの独自問題 | 97問（4.8%） |

最頻出クラスタは17期全てに出題（揚力・迎え角・アドバースヨー・航法ログ計算など）。

## 勉強戦略への示唆

1. **直近2年分（4期）を解法まで理解する**のが最効率。それ以前は伸びが緩やか
2. **航法の計算問題（航法ログ・風力三角形など）は毎回数値が変わる**ため丸暗記では取れない。解法の習得が必須（逆に解法さえ身につけば航法は既視率99%）
3. 丸暗記派は**通信・工学（回転翼）**の初見率が高めなので注意
4. 数字は「見た問題は必ず正解できる」前提の上限値。実際の得点は理解度に依存
