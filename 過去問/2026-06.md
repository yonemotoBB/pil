**例題１**
耐空性審査要領における定義で誤りはどれか。
- [ ] （１）「制限荷重」とは、常用運用状態において予想される最大の荷重をいう。
- [ ] （２）「ホバリング」とは、回転翼航空機にあって、対気速度零の飛行状態をいう。
- [ ] （３）「自動回転飛行」とは、回転翼航空機が運動中、その揚力を受持つ回転翼が完全に空力のみ
によって駆動される飛行状態をいう。
- [ ] （４） ｢ＶＹ｣とは､安全離陸速度をいう。
:::spoiler 正答
４

```
用語: 耐空性審査要領＝航空機の耐空証明検査の基準を定めた国の要領。
① 制限荷重＝常用運用状態で予想される最大の荷重 … 正。これを超える強度余裕として終極荷重（制限荷重×安全率1.5）がある。
② ホバリング＝回転翼航空機の対気速度零の飛行状態 … 正。対地速度ではなく「対気」速度零である点に注意。
③ 自動回転飛行＝揚力を受け持つ回転翼が完全に空力のみで駆動される飛行状態 … 正。エンジン動力を用いないオートローテーションの定義そのもの。
④ ＶＹ＝安全離陸速度 … 誤。ＶＹは「最良上昇率速度」（単位時間あたりの上昇高度が最大となる速度）。安全離陸速度はＶＴＯＳＳ。なおＶＸは最良上昇角速度。
補足: ＶＸ＝距離あたりの上昇（角度）最大、ＶＹ＝時間あたりの上昇（率）最大、と対で覚える。
```

**例題２**
耐空類別が回転翼航空機普通Ｎの対気速度計の標識について、赤色放射線が表す意味で正しいも
のはどれか。
- [ ] （１）最大運用限界速度
- [ ] （２）最大巡航速度
- [ ] （３）超過禁止速度
- [ ] （４）失速速度
:::spoiler 正答
３

```
用語: 耐空類別 回転翼航空機普通Ｎ＝最大離陸重量等が一定以下の普通カテゴリのヘリコプタ。対気速度計の標識＝計器の弧・線の色で速度範囲や限界を示すもの。
① 最大運用限界速度 … 誤。ＶＭＯは主に高速の飛行機（輸送類別等）で用いられる用語で、赤色放射線の意味ではない。
② 最大巡航速度 … 誤。巡航速度域は緑色弧線などの範囲標識で示され、赤色放射線ではない。
③ 超過禁止速度 … 正。赤色放射線はＶＮＥ（超過禁止速度）を示し、いかなる場合もこれを超えてはならない。
④ 失速速度 … 誤。ヘリコプタには固定翼機のような失速速度の概念がなく、赤色放射線の意味でもない。
補足: 緑色弧線＝常用運用範囲、黄色弧線＝警戒範囲、赤色放射線＝超過禁止速度（ＶＮＥ）と整理して覚える。
```

**例題３**
ロータ・ブレードの風圧中心が前方へ移動するのは次のうちどれか。
- [ ] （１）水平飛行のとき
- [ ] （２）迎え角を大きくしたとき
- [ ] （３）飛行速度を増加したとき
- [ ] （４）迎え角を小さくしたとき
:::spoiler 正答
２

```
用語: 風圧中心＝翼に働く空気力の合力の作用点。迎え角＝翼弦線と相対風のなす角。
① 水平飛行のとき … 誤。水平飛行という飛行状態そのものは風圧中心の移動を決めない。移動を決めるのは迎え角の変化である。
② 迎え角を大きくしたとき … 正。キャンバーを持つ翼型では迎え角を増すと風圧中心は前方（前縁側）へ移動する。
③ 飛行速度を増加したとき … 誤。速度増加そのものではなく、それに伴う迎え角の減少があれば風圧中心はむしろ後方へ移動する。
④ 迎え角を小さくしたとき … 誤。迎え角を小さくすると風圧中心は後方へ移動する。前方移動とは逆。
補足: 「迎え角 大 → 風圧中心 前へ、迎え角 小 → 後ろへ」と対で覚える。
```

**例題４**
定常ドラッギングに関する記述について誤りはどれか。
- [ ] （１）ブレードが中立位置より進む角度をリード角、遅れる角度をラグ角という。
- [ ] （２）エンジン始動時、ブレードは中立位置より後方に遅れる。
- [ ] （３）オートローテーションのときは、ブレードは空気力によって駆動されるのでブレードは中
立位置より前方向に進む。
- [ ] （４）回転数が高いときは、遠心力が小さくなるので遅れ角は大きくなる。
:::spoiler 正答
４

```
用語: ドラッギング＝ブレードが回転面内で前後（リード・ラグ）に振れる運動。リード角＝中立位置より進む角度、ラグ角＝遅れる角度。
① リード角・ラグ角の定義 … 正。回転面内で進む側がリード、遅れる側がラグである。
② エンジン始動時は中立位置より後方に遅れる … 正。駆動トルクがブレードの慣性・空気抵抗に打ち勝って回すため、ブレードはラグ（遅れ）側に振れる。
③ オートローテーション時は前方に進む … 正。空気力がブレードを駆動する側に働くため、ブレードはリード（進み）側に振れる。
④ 回転数が高いと遠心力が小さくなり遅れ角が大きくなる … 誤。回転数が高いほど遠心力は「大きく」なり、ブレードを半径方向外側へ張る力が強まるため遅れ角は「小さく」なる。因果が両方とも逆。
補足: 遠心力は回転数の2乗に比例して増大する。遠心力大→ブレードが張られて振れにくい、と物理的に理解する。
```

**例題５**
安定性に関する記述で誤りはどれか。
- [ ] （１）静的安定性とは、ヘリコプタがトリム状態にあるとき外乱を受けて角変位または速度変化
があった後、初期傾向としてトリム状態に戻ろうとするか、あるいは離れようとするかとい
う性質をいう。
- [ ] （２）動的安定性とは、静的に安定なヘリコプタがトリム状態にあるとき外乱を受けて角変位ま
たは速度変化があった後、時間の経過とともにトリム状態に戻るか、あるいは離れてしまう
かという性質をいう。
- [ ] （３）外乱を受け角変位または速度変化があった後、その変化した状態のままに止まっていよう
とする傾向にある場合、そのヘリコプタは静的に安定であるという。
- [ ] （４）外乱を受け角変位または速度変化があった後、トリム状態から離れようとする初期傾向が
ある場合、そのヘリコプタは静的に不安定であるという。
:::spoiler 正答
３

```
用語: 静的安定＝外乱後の「初期傾向」としてトリム状態（釣り合い状態）に戻ろうとする性質。動的安定＝「時間の経過とともに」トリム状態に戻るかどうかの性質。
① 静的安定性の定義（初期傾向で戻るか離れるか） … 正。
② 動的安定性の定義（時間経過で戻るか離れるか） … 正。
③ 変化した状態のままに止まる傾向がある場合を静的に安定という … 誤。変化した状態に止まる傾向は「静的中立（中立安定）」であり、静的安定ではない。静的安定は元に戻ろうとする初期傾向を持つ場合をいう。
④ トリム状態から離れようとする初期傾向がある場合は静的に不安定 … 正。
補足: 静的安定＝戻る、静的中立＝そのまま、静的不安定＝離れる、の3区分。静的は「初期傾向」、動的は「時間経過後の挙動」がキーワード。
```

**例題６**
メイン・ロータ・ブレードの捩り下げに関する記述で誤りはどれか。
- [ ] （１）翼端失速を遅らせる。
- [ ] （２）通常、8°～14°の範囲の捩り下げが使用される。
- [ ] （３）メイン・ロータの回転数を一定に保ちやすくする。
- [ ] （４）ホバリング時にロータ効率を向上させる効果がある。
:::spoiler 正答
３

```
用語: 捩り下げ（ねじり下げ）＝ブレードの取付角を翼根で大きく翼端で小さくすること。翼端失速＝翼端付近が先に失速する現象。
① 翼端失速を遅らせる … 正。翼端の迎え角を小さくしておくことで、前進飛行時の後退側ブレード翼端などの失速を遅らせる。
② 通常8°～14°の範囲の捩り下げが使用される … 正（本例題集の記述による）。
③ メイン・ロータの回転数を一定に保ちやすくする … 誤。捩り下げは翼幅方向の揚力（誘導速度）分布を均一化するためのもので、回転数の保持とは関係がない。回転数の維持はガバナやコレクティブ操作によるもの。
④ ホバリング時にロータ効率を向上させる … 正。誘導速度分布が均一に近づくことで誘導損失が減り、ホバリング効率が上がる。
補足: 捩り下げの目的は「揚力分布の均一化→効率向上」と「翼端失速の抑制」の2点、と整理する。
```

**例題７**
トルク効果に関する記述で正しいものはどれか。
- [ ] （１）メイン・ロータを駆動するエンジンのトルク
- [ ] （２）胴体がメイン・ロータの回転方向と同じ方向に回転しようとする効果
- [ ] （３）テール・ロータを駆動するトランスミッションのトルク
- [ ] （４）胴体がメイン・ロータの回転方向と反対方向に回転しようとする効果
:::spoiler 正答
４

```
用語: トルク効果＝メイン・ロータを駆動する反作用（反トルク）で胴体が回されようとする現象。
① メイン・ロータを駆動するエンジンのトルク … 誤。それは駆動トルクそのものであり、「トルク効果」は駆動の反作用として機体に現れる現象を指す。
② 胴体がロータの回転方向と同じ方向に回転しようとする … 誤。作用・反作用の法則により、胴体はロータ回転方向と「反対」方向に回されようとする。
③ テール・ロータを駆動するトランスミッションのトルク … 誤。テール・ロータはトルク効果を打ち消すための装置であり、トルク効果の定義ではない。
④ 胴体がロータの回転方向と反対方向に回転しようとする … 正。この反トルクを打ち消すためにテール・ロータの推力（アンチトルク）が必要となる。
補足: 「ロータを右に回せば胴体は左に回されようとする」— 作用・反作用で覚える。
```

Ｈ－Ｖ線図（高度－速度包囲線図）に関する説明で誤りはどれか。
- [ ] （１）Ｈ－Ｖ線図は、動力装置故障状態で安全に着陸することができない高度－前進速度の組み
合わせを示したものである。
- [ ] （２）Ｈ－Ｖ線図の縦軸は地表面からスキッド（降着装置）までの対地高度、横軸は指示対気速

- [ ] （３）Ｈ－Ｖ線図の飛行回避領域は、ヘリコプタの全備重量が大きくなると危険域が縮小する。
- [ ] （４）Ｈ－Ｖ線図は、平均的なパイロットの操縦技術と反応時間を想定して作成される。

３

```

④ 平均的なパイロットの操縦技術と反応時間を想定して作成 … 正。
補足: 重量増・高密度高度（高温・高標高）はいずれも性能を悪化させ、回避領域を拡大させる方向に働く。

**例題９**
利用馬力、必要馬力及び余剰馬力に関する説明で誤りはどれか。
- [ ] （１）ヘリコプタが実際に利用できる馬力を利用馬力という。
- [ ] （２）利用馬力は、トランスミッション定格やエンジン性能により制限される。
- [ ] （３）利用馬力と必要馬力の差を余剰馬力という。
- [ ] （４）余剰馬力は、ヘリコプタの対気速度により変化しない。
:::spoiler 正答
４

```
用語: 利用馬力＝機体が実際に利用できる馬力。必要馬力＝その飛行状態の維持に必要な馬力。余剰馬力＝利用馬力－必要馬力で、上昇や加速に使える余力。
① 実際に利用できる馬力を利用馬力という … 正。
② 利用馬力はトランスミッション定格やエンジン性能で制限される … 正。エンジン出力が大きくても伝達系統の定格を超えては使えない。
③ 利用馬力と必要馬力の差が余剰馬力 … 正。
④ 余剰馬力は対気速度により変化しない … 誤。必要馬力は対気速度により大きく変化する（ホバリングで大、中速域で最小、高速で再び増大）ため、その差である余剰馬力も速度により変化する。余剰馬力が最大となる速度付近が最良上昇率速度ＶＹとなる。
補足: 必要馬力曲線はバケツ型（U字型）。曲線の底＝必要馬力最小の速度で余剰馬力最大、と関連づけて覚える。
```

**例題 10**
オートローテーション時のブレードの各領域に関する記述で正しいものはどれか。
- [ ] （１）プロペラ領域は、ブレードを減速させる。
- [ ] （２）オートローテーション領域は、ブレードを減速させる。
- [ ] （３）失速領域は、ブレードを加速させる。
- [ ] （４）ロータ領域は、ブレードを加速させる。
:::spoiler 正答
１

```
用語: オートローテーション時のブレードは半径方向に3領域に分かれる。プロペラ領域（翼端側）＝空気力がブレードを減速させる領域。オートローテーション領域（駆動領域、中間部）＝空気力がブレードを加速（駆動）させる領域。失速領域（翼根側）＝迎え角過大で失速し抗力でブレードを減速させる領域。
① プロペラ領域はブレードを減速させる … 正。翼端側では合成空気力が回転を妨げる向きに働く。
② オートローテーション領域はブレードを減速させる … 誤。この領域（駆動領域）では合成空気力が回転面より前に傾き、ブレードを「加速（駆動）」する。
③ 失速領域はブレードを加速させる … 誤。翼根側の失速領域は抗力が大きく、ブレードを「減速」させる。
④ 「ロータ領域」 … 誤。そのような領域区分は存在しない。
補足: 翼端から順に「プロペラ（減速）→駆動（加速）→失速（減速）」。定常オートローテーションでは加速と減速が釣り合い回転数が一定に保たれる。
```

**例題 11**
セットリング ウィズ パワーに関する説明で誤りはどれか。
- [ ] （１）ボルテックスリングは非定常に変動するため、トルクの変動と共にサイクリックのコント
ロールが困難となる。
- [ ] （２）セットリング ウィズ パワーに陥りやすい飛行状態としては、地面効果外ホバリング、低
速高角度進入、低速飛行時の不用意な降下がある。
- [ ] （３）セットリング ウィズ パワーに陥った場合の兆候のひとつに、機体及びサイクリックステ
ィックの振動がある。
- [ ] （４）完全なセットリング ウィズ パワーに入った場合は、回復に約 100ft 程度の高度が必要と
なる。
:::spoiler 正答
４

```
用語: セットリング ウィズ パワー（ボルテックス・リング状態）＝動力を使用したまま自らの吹き下ろし（ダウンウォッシュ）の中に沈み込み、ロータ周りに渦輪が形成されて推力が失われ、パワーを増しても降下が止まらなくなる状態。
① ボルテックスリングが非定常に変動しトルク変動と共にサイクリック操縦が困難になる … 正。渦の変動により機体の動揺・操縦性低下が生じる。
② 陥りやすい飛行状態＝地面効果外ホバリング、低速高角度進入、低速飛行時の不用意な降下 … 正。目安として「低速（前進速度小）＋大きな降下率＋パワー使用中」の組合せが危険。
③ 兆候のひとつに機体及びサイクリックスティックの振動がある … 正。
④ 完全に入った場合の回復に約100ft程度の高度が必要 … 誤。完全なセットリング ウィズ パワーからの回復には数百フィート以上の高度損失を伴い、100ft程度では回復できない。低高度で陥ると回復不能となるおそれがある。
補足: 回復操作は前進速度をつけて渦輪から抜け出すこと（コレクティブを単に増すと悪化する）。
```

**例題 12**
ホバリングを含む低速飛行中のヘリコプタにヨー方向の不安定をもたらす領域で誤りはどれか。
- [ ] （１）地面効果による不安定領域
- [ ] （２）メイン・ロータ・ディスク・ボルテックスの影響による不安定領域
- [ ] （３）テール・ロータのボルテックス・リング状態による不安定領域
- [ ] （４）風見安定による不安定領域
:::spoiler 正答
１

```
